# 📌 프로젝트 기획서 (1 Page)

## 🎯 프로젝트 이름

**DataNexus: 영상·텍스트 기반 검색 & 질문 시스템**

---

## 💡 한 줄 설명

유튜브 영상, 문서(PDF/txt), 로그 데이터를 업로드하면
👉 **검색 + 질문(Q&A)** 으로 원하는 내용을 빠르게 찾을 수 있는 시스템

---

## 🔥 문제 정의

* 영상은 원하는 구간 찾기 어려움
* 문서/로그는 검색은 되지만 “이해”는 어려움
* 데이터가 많아질수록 탐색 비용 증가

👉 해결:
**“검색 + 질문”을 결합한 탐색 시스템**

---

## 🧠 핵심 기능

### 1️⃣ 데이터 입력

* 유튜브 링크 (STT)
* PDF / txt 파일
* 로그 데이터

---

### 2️⃣ 처리 파이프라인

```plaintext
데이터 입력
→ 텍스트 추출 (STT / 파싱)
→ chunk 분할
→ embedding 생성
→ vector DB 저장
```

---

### 3️⃣ 사용자 기능

#### 🔍 검색

* 키워드 기반
* 타임스탬프 이동 (영상)

#### 💬 질문 (Q&A)

* 자연어 질문 입력
* 관련 구간 자동 추출
* 근거 기반 답변 제공

---

## 🖥️ 시스템 구조

```plaintext
Frontend (UI)
- 검색창 / 질문 입력
- 결과 표시

Backend (FastAPI)
- 파일 업로드
- 검색 / Q&A API

AI Layer
- Whisper (STT)
- Embedding
- Vector DB (FAISS/Chroma)
- LLM (응답 생성)
```
